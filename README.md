# Techno AI Marketing & IT — One‑Page Site (Vite + React + Tailwind)

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to Vercel
1. Push this project to GitHub.
2. Go to vercel.com → New Project → Import repo.
3. Framework preset: **Vite**.
4. Build command: `vite build`, Output: `dist`.
5. Click Deploy.
